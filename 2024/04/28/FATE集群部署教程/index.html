2.2 关闭selinux

SELinux（Security-Enhanced Linux）是一种安全机制和强制访问控制（MAC）系统，用于增强 Linux 操作系统的安全性。

在目标服务器目标服务器（192.168.40.128 192.168.40.132 ）root用户下执行：

确认是否已安装selinux

centos系统执行：rpm -qa | grep selinux

如果已安装了selinux就执行：setenforce 0

setenforce 0 是一个用于临时禁用 SELinux 的命令。执行该命令会将 SELinux 模式更改为 “Permissive” 模式，也就是 SELinux 仍然处于启用状态，但不会阻止或拒绝任何操作。这条指令仅在当前会话中生效。重新启动服务器或重新加载 SELinux 时，它会恢复为之前的状态。
