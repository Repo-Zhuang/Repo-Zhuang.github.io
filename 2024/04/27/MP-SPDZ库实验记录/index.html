1. 环境变量

  • /usr/local/bin：这个目录用于存放用户自行安装的软件的可执行文件。通常，用户可以将自己编译或安装的软件安装到 /usr/local 目录下，这样可以与系统自带的软件进行区分。例如，当你使用命令 make install 安装某个软件时，默认情况下，可执行文件会被复制到 /usr/local/bin 目录中。

  • /usr/bin：这个目录是系统默认的二进制可执行文件的存放目录。在 Linux 系统中，大部分的命令和工具都安装在 /usr/bin 目录下。这些命令和工具是系统所提供的，通常随着操作系统的安装而自动包含在其中。

  • 当在命令行中输入一个命令时，系统会在 $PATH 环境变量中指定的目录中查找可执行文件，对于某些系统和发行版，/usr/local/bin 可能不在默认的 $PATH 中。
